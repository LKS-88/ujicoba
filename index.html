Informasi Aplikasi

Panduan Penggunaan

Berikut adalah panduan penggunaan aplikasi BARJAS (Sistem Bantuan Terpadu).
Cara Input Data
  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Untuk nomor WhatsApp, masukkan tanpa kode negara (+62)
  4. Pilih desa dari dropdown atau ketikkan jika tidak ada
  5. Pilih tahun pemberian bantuan
  6. Pilih jenis bantuan atau ketikkan jenis baru
  7. Masukkan jumlah bantuan (dalam satuan)
  8. Tambahkan link Google Drive jika ada dokumen pendukung
  9. Klik Simpan Data untuk menyimpan
Ekspor Data

Anda dapat mengekspor data dalam format:

  • Excel - Untuk analisis data lebih lanjut
  • PDF - Untuk laporan dan dokumentasi

Caranya:

  1. Buka menu Data Penerima
  2. Gunakan filter jika perlu memfilter data tertentu
  3. Klik tombol Excel atau PDF
  4. File akan otomatis terunduh
Filter dan Pencarian

Anda dapat memfilter data berdasarkan:

  • Desa - Menampilkan data dari desa tertentu
  • Jenis Bantuan - Menampilkan data berdasarkan jenis bantuan
  • Tahun - Menampilkan data dari tahun tertentu

Anda juga dapat menggunakan fitur pencarian untuk mencari data berdasarkan nama penerima.

Setelah memilih filter, klik Terapkan Filter untuk melihat hasil.

Statistik dan Grafik

Menu Statistik menampilkan visualisasi data dalam bentuk grafik dan chart, termasuk:

  • Grafik batang distribusi bantuan per desa
  • Diagram pie distribusi jenis bantuan
  • Grafik line trend bantuan per tahun
  • Data statistik ringkasan

Data akan diperbarui otomatis saat ada perubahan data.

Keamanan

Untuk keamanan data, aplikasi dilindungi dengan kode akses.

Kode akses default adalah 1234.

Anda dapat mengubah kode akses melalui menu Pengaturan > Pengaturan Keamanan.

Jangan bagikan kode akses kepada orang yang tidak berwenang.
