Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Pastikan NIK terdiri dari 16 digit angka
  4. Pilih desa dari dropdown yang tersedia
  5. Pilih tahun dari opsi 2020-2090
  6. Klik tombol Simpan Data

  1. Buka menu Data Bantuan
  2. Gunakan filter untuk menyaring data berdasarkan tahun, jenis bantuan, atau desa
  3. Gunakan kotak pencarian untuk mencari data berdasarkan nama atau NIK
  4. Gunakan pagination untuk navigasi halaman data

  1. Buka menu Laporan
  2. Pilih tahun, jenis bantuan, dan desa yang ingin dilaporkan
  3. Pilih format laporan (PDF atau Excel)
  4. Klik tombol Generate Laporan
  5. Tunggu hingga proses selesai dan laporan akan diunduh secara otomatis
FAQ (Pertanyaan Umum)

Untuk mengubah data yang sudah terinput, buka menu Data Bantuan, cari data yang ingin diubah, lalu klik tombol Edit pada kolom aksi. Setelah selesai mengubah data, klik tombol Simpan Perubahan.

Data yang sudah diinput akan otomatis tersimpan di browser Anda. Namun, disarankan untuk selalu melakukan backup data secara berkala dengan menggunakan fitur Backup Data di menu Pengaturan.

Untuk mencetak data individual, buka menu Data Bantuan, cari data yang ingin dicetak, lalu klik tombol Cetak pada kolom aksi. Data akan dicetak dalam format yang rapi dan siap digunakan.
