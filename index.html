Bantuan & Panduan Penggunaan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Pastikan data yang dimasukkan sudah benar
  4. Klik tombol Simpan Data

Pada menu Data Penerima, gunakan filter yang tersedia:

  • Filter berdasarkan Kecamatan
  • Filter berdasarkan Desa
  • Filter berdasarkan Status Verifikasi
  • Gunakan kotak pencarian untuk mencari data tertentu

Pada menu Data Penerima, gunakan tombol:

  • Export Excel untuk mengekspor data ke format Excel
  • Export PDF untuk mengekspor data ke format PDF
  • Cetak untuk mencetak data langsung
Pengaturan Keamanan

Untuk mengubah kode keamanan aplikasi:

  1. Buka menu Pengaturan
  2. Masukkan kode keamanan saat ini
  3. Masukkan kode keamanan baru
  4. Konfirmasi kode keamanan baru
  5. Klik tombol Simpan Perubahan
Simpan kode keamanan di tempat yang aman dan jangan berikan kepada orang lain.
