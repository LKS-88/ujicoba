Bantuan dan Panduan

Tentang Aplikasi
BARJAS - Sistem Bantuan Terpadu

Aplikasi ini dirancang untuk membantu dalam pendataan dan manajemen penerima bantuan di Kabupaten Situbondo.

Fitur Utama
  • Input data penerima bantuan dengan validasi
  • Manajemen data penerima (lihat, edit, hapus)
  • Statistik dan visualisasi data
  • Ekspor data ke berbagai format (Excel, PDF, CSV)
  • Pencetakan laporan
  • Backup dan restore data
  • Keamanan dengan kode akses
Versi

v2.0.1 - Rilis stabil

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Pastikan data yang dimasukkan valid
  4. Klik tombol Simpan Data
  5. Data akan tersimpan dan muncul di tabel data

  1. Buka menu Laporan
  2. Pilih jenis laporan yang diinginkan
  3. Atur filter sesuai kebutuhan (desa, jenis bantuan, atau periode)
  4. Klik tombol Cetak Laporan
  5. Laporan akan dicetak dalam format landscape

  1. Buka menu Laporan
  2. Pilih format ekspor (Excel, PDF, atau CSV)
  3. Tentukan data yang akan diekspor (semua, terfilter, atau terpilih)
  4. Klik tombol Ekspor Data
  5. File akan diunduh ke perangkat Anda

Keamanan

Untuk keamanan data Anda:

  • Gunakan kode keamanan yang kuat
  • Lakukan backup data secara berkala
  • Jangan bagikan kode keamanan kepada orang lain
  • Pastikan perangkat Anda terlindungi password
