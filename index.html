Tentang Aplikasi

Informasi Aplikasi
APLIKASI BARJAS (Bantuan Sosial)

Aplikasi ini dirancang untuk membantu dalam pengelolaan data bantuan sosial secara digital. Dengan aplikasi ini, Anda dapat:

  • Mencatat data penerima bantuan dengan lengkap
  • Mengelola data bantuan yang telah didistribusikan
  • Mencetak laporan dalam format PDF
  • Mengekspor data ke format Excel
  • Memfilter data berdasarkan kriteria tertentu

Aplikasi ini menggunakan teknologi modern berbasis web yang dapat diakses dari berbagai perangkat.

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Klik tombol Simpan Data untuk menyimpan
  4. Data akan tersimpan dan dapat dilihat di menu Data Bantuan

  1. Buka menu Laporan
  2. Pilih jenis laporan yang diinginkan
  3. Klik Pratinjau Laporan untuk melihat tampilan sebelum cetak
  4. Klik Cetak ke PDF untuk mengunduh laporan dalam format PDF

  1. Buka menu Data Bantuan
  2. Gunakan filter yang tersedia di bagian atas tabel
  3. Pilih tahun, kecamatan, atau jenis bantuan yang ingin ditampilkan
  4. Gunakan kotak pencarian untuk mencari data tertentu
  5. Klik Terapkan Filter untuk menerapkan filter
