Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan asterisk *)
  3. Pastikan NIK terdiri dari 16 digit angka
  4. Pastikan nomor WhatsApp valid
  5. Pilih kecamatan dan desa sesuai dengan data penerima
  6. Pilih jenis dan jumlah bantuan yang diberikan
  7. Klik tombol Simpan Data untuk menyimpan

Untuk melihat data yang sudah diinput, buka menu Data Bantuan. Anda dapat:

  • Menggunakan filter tahun, kecamatan, atau jenis bantuan
  • Mencari data tertentu menggunakan kolom pencarian
  • Mengurutkan data dengan mengklik header kolom
  • Memilih beberapa data untuk dilakukan aksi massal

Untuk membuat laporan, buka menu Laporan. Anda dapat:

  • Mengekspor data ke format Excel untuk analisis lebih lanjut
  • Membuat laporan PDF yang dapat dicetak
  • Mencetak laporan langsung dari browser
  • Melihat riwayat ekspor yang pernah dilakukan
