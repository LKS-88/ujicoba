Informasi Aplikasi
APLIKASI BARJAS (Sistem Informasi Bantuan Sosial)

Aplikasi ini dirancang untuk membantu dalam pendataan dan pengelolaan bantuan sosial yang disalurkan kepada masyarakat. Dengan aplikasi ini, Anda dapat:

  • Mencatat data penerima bantuan
  • Mengelola informasi jenis dan jumlah bantuan
  • Menganalisis distribusi bantuan per desa
  • Membuat laporan dalam format Excel dan PDF
  • Mencetak data untuk keperluan dokumentasi

Aplikasi ini dibuat dengan menggunakan teknologi HTML, CSS, JavaScript, Bootstrap, dan library Chart.js untuk visualisasi data.

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan asterisk)
  3. Pastikan data yang dimasukkan sudah benar
  4. Klik tombol Simpan Data untuk menyimpan

  1. Buka menu Data Bantuan untuk melihat semua data
  2. Gunakan filter untuk menyaring data berdasarkan kriteria tertentu
  3. Buka menu Laporan untuk melihat analisis dan statistik data
  4. Gunakan tombol ekspor untuk mengunduh data dalam format Excel atau PDF

  3. Klik tombol Cetak untuk mencetak data yang ditampilkan
