Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang diperlukan (ditandai dengan asterisk *)
  3. Klik tombol Simpan Data untuk menyimpan
  4. Data akan tersimpan dan dapat dilihat di menu Data Penerima

Anda dapat memfilter data berdasarkan:

  • Desa - menggunakan menu Filter Data
  • Jenis Bantuan - menggunakan menu Filter Data
  • Tahun - menggunakan menu Filter Data
  • Tanggal - menggunakan rentang tanggal di menu Filter Data
  • Kata kunci - menggunakan kotak pencarian di menu Data Penerima

Untuk mengekspor data:

  1. Buka menu Ekspor Data
  2. Pilih format ekspor (Excel, PDF, CSV, atau Cetak)
  3. Tentukan rentang data yang akan diekspor (Semua, Terfilter, atau Terpilih)
  4. Pilih kolom yang ingin disertakan dalam ekspor
  5. Klik tombol ekspor sesuai format yang diinginkan

Backup Data:

  1. Buka menu Backup & Restore
  2. Klik tombol Backup Data
  3. File backup akan terunduh secara otomatis

Restore Data:

  1. Buka menu Backup & Restore
  2. Klik Pilih File dan pilih file backup sebelumnya
  3. Klik tombol Restore Data
  4. Data akan diganti dengan data dari file backup
Pemecahan Masalah

Pastikan semua field wajib telah diisi dengan benar. Field wajib ditandai dengan asterisk (*) dan akan menampilkan pesan error jika tidak diisi dengan benar.

Jika aplikasi berjalan lambat, coba:

  • Kurangi jumlah data yang ditampilkan per halaman
  • Gunakan filter untuk membatasi data yang ditampilkan
  • Bersihkan cache browser
