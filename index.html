Bantuan dan Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Klik tombol Generate untuk membuat kode validasi
  4. Klik tombol Simpan Data untuk menyimpan
  5. Data akan tersimpan di database aplikasi

  1. Buka menu Lihat Data
  2. Gunakan filter untuk mempersempit pencarian
  3. Gunakan kotak pencarian untuk mencari data tertentu
  4. Klik tombol aksi untuk melihat detail, edit, atau hapus data

Backup Data:

  1. Buka menu Backup/Restore
  2. Klik tombol Backup Data ke JSON
  3. File backup akan diunduh ke perangkat Anda

Restore Data:

  1. Buka menu Backup/Restore
  2. Klik Pilih File dan pilih file backup
  3. Klik tombol Restore Data dari JSON
  4. Data akan dipulihkan dari file backup
