Informasi Aplikasi

Selamat datang di Aplikasi BARJAS (Bank Sampah) Dinas Peternakan dan Perikanan Situbondo.

Aplikasi ini digunakan untuk mengelola data sampah yang terkumpul dari berbagai sumber. Fitur-fitur yang tersedia:

  • Input data sampah dengan lokasi GPS
  • Melihat dan mengelola data sampah
  • Menyimpan link Google Drive untuk dokumentasi
  • Mencetak laporan data sampah
  • Ekspor data ke Excel dan PDF

Gunakan menu di sebelah kiri untuk mengakses berbagai fitur yang tersedia.
