Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Pastikan NIK terdiri dari 16 digit angka
  4. Pastikan nomor WhatsApp diawali dengan kode negara (contoh: 628123456789)
  5. Klik tombol Simpan Data untuk menyimpan

  1. Buka menu Data Bantuan
  2. Gunakan field pencarian untuk mencari berdasarkan nama, NIK, atau alamat
  3. Gunakan filter berdasarkan jenis bantuan, kecamatan, atau desa
  4. Klik tombol Filter untuk menerapkan filter
  5. Klik tombol Reset untuk menghapus semua filter

  1. Buka menu Laporan
  2. Pilih jenis laporan yang diinginkan
  3. Isi parameter laporan sesuai kebutuhan
  4. Pilih format laporan (PDF atau Excel)
  5. Klik tombol Unduh Laporan untuk mengunduh
