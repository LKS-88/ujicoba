Tentang Aplikasi

BARJAS (Sistem Bantuan Terpadu) adalah aplikasi untuk mengelola data bantuan yang disalurkan oleh Dinas Peternakan dan Perikanan Kabupaten Situbondo.

Versi: 9.8.96

Dikembangkan oleh: Lentera Karya Situbondo

Tanggal Rilis: September 2025

Panduan Cepat
  1. Gunakan menu Input Data untuk menambahkan data penerima bantuan baru
  2. Gunakan menu Data Bantuan untuk melihat, mengedit, atau menghapus data
  3. Gunakan menu Laporan untuk melihat statistik dan ekspor data
  4. Gunakan menu Pengaturan untuk mengatur preferensi aplikasi
FAQ (Pertanyaan Umum)

Pergi ke menu Input Data, isi formulir yang disediakan, lalu klik tombol Simpan Data.

Di menu Data Bantuan, gunakan tombol Cetak Data di bagian bawah tabel.

Pergi ke menu Pengaturan > tab Keamanan, lalu masukkan kode baru dan konfirmasi.
