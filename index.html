Panduan Penggunaan Aplikasi

Panduan Lengkap

Dashboard menampilkan ringkasan data bantuan yang telah dimasukkan. Anda dapat melihat:

  • Total penerima bantuan
  • Total dana yang telah disalurkan
  • Jumlah desa yang telah terjangkau
  • Rata-rata bantuan per penerima
  • Grafik distribusi bantuan per desa
  • Grafik jenis bantuan
  • Data terbaru yang dimasukkan

Gunakan tombol Refresh untuk memperbarui data dashboard.

Form input data digunakan untuk menambahkan data penerima bantuan baru. Pastikan mengisi semua field yang wajib diisi:

  • Nama Penerima - Nama lengkap penerima bantuan
  • NIK - Nomor Induk Kependudukan (16 digit)
  • Nomor WhatsApp - Nomor WhatsApp aktif penerima (tanpa kode negara +62)
  • Alamat - Alamat lengkap penerima
  • Desa - Desa tempat tinggal penerima
  • Jenis Bantuan - Jenis bantuan yang diterima
  • Jumlah Bantuan - Nominal bantuan dalam Rupiah
  • Tanggal Penerimaan - Tanggal bantuan diberikan
  • Kode Validasi - Kode keamanan untuk validasi data

Gunakan tombol Generate Kode untuk menghasilkan kode validasi otomatis.

Gunakan tombol Preview untuk melihat pratinjau data sebelum disimpan.

Tabel data menampilkan semua data bantuan yang telah dimasukkan. Anda dapat:

  • Melihat semua data dalam format tabel
  • Menyaring data berdasarkan desa, jenis bantuan, atau rentang tanggal
  • Mencari data berdasarkan kata kunci
  • Mengedit data yang sudah ada
  • Menghapus data yang tidak diperlukan
  • Melihat dokumen pendukung melalui link Google Drive
  • Menghubungi penerima via WhatsApp
  • Mencetak, mengekspor ke Excel, atau mengekspor ke PDF

Gunakan fitur filter dan pencarian untuk menemukan data tertentu dengan cepat.

Halaman grafik dan statistik menampilkan visualisasi data bantuan. Anda dapat melihat:

  • Trend Bantuan per Bulan - Grafik perkembangan bantuan dari waktu ke waktu
  • Distribusi per Desa - Diagram pie menunjukkan persentase bantuan per desa
  • Statistik Deskriptif - Rata-rata, tertinggi, terendah, dan total data
  • Perbandingan Jenis Bantuan - Grafik perbandingan antar jenis bantuan per desa

Gunakan dropdown selector untuk mengubah jenis grafik perbandingan (Bar, Line, atau Radar).

Halaman pengaturan memungkinkan Anda mengkonfigurasi aplikasi sesuai kebutuhan:

  • Ubah Kode Keamanan - Mengganti kode akses aplikasi
  • Reset Data - Menghapus semua data yang tersimpan
  • Ekspor/Impor Data - Backup dan restore data
  • Info Aplikasi - Informasi versi dan pengembang

Simpan kode keamanan baru di tempat yang aman untuk mencegah kehilangan akses.

Pertanyaan Umum

Untuk mengubah kode keamanan, buka halaman Pengaturan dan pilih opsi Ubah Kode Keamanan. Masukkan kode lama dan kode baru yang diinginkan, kemudian simpan.

Ya, data Anda disimpan secara lokal di browser Anda dan tidak dikirim ke server manapun. Untuk keamanan tambahan, gunakan fitur ekspor data untuk membuat backup secara berkala.

Untuk mencetak data, buka halaman Tabel Data Bantuan, gunakan filter jika perlu, lalu klik tombol Cetak. Browser akan membuka jendela cetak dengan layout yang sudah dioptimalkan.

Saat ini, daftar desa sudah ditetapkan. Jika perlu menambahkan desa baru, hubungi administrator sistem untuk mengubah kode aplikasi.
