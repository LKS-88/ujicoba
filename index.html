Informasi Aplikasi

BARJAS adalah aplikasi untuk mengelola data penerima bantuan dana sosial secara terpadu.

Fitur yang tersedia:

  • Input data penerima bantuan
  • Manajemen data penerima
  • Verifikasi data
  • Ekspor data ke Excel dan PDF
  • Cetak data per penerima

Panduan Penggunaan

Cara Menggunakan Aplikasi
Input Data

1. Buka menu Input Data

2. Isi semua field yang wajib diisi (ditandai dengan *)

3. Klik tombol Simpan Data untuk menyimpan

Melihat Data

1. Buka menu Data Penerima

2. Gunakan filter untuk menyaring data sesuai kebutuhan

3. Gunakan tombol ekspor untuk mengunduh data

Pengaturan

1. Buka menu Pengaturan

2. Ubah kode keamanan jika diperlukan

3. Lakukan backup data secara berkala

Troubleshooting
Data tidak tersimpan

Pastikan semua field wajib telah diisi dengan benar

Filter tidak bekerja

Pastikan telah memilih kriteria filter dan menekan tombol Terapkan Filter

Ekspor data tidak berfungsi

Pastikan browser mendukung fitur ekspor dan tidak memblokir popup

Lupa kode keamanan

Hubungi administrator untuk reset kode keamanan
