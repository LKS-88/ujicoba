Panduan Penggunaan

Panduan Aplikasi BARJAS

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Pastikan NIK belum terdaftar sebelumnya
  4. Klik tombol Generate untuk membuat kode validasi
  5. Klik Simpan Data untuk menyimpan data

  1. Buka menu Data Penerima
  2. Gunakan filter desa atau status untuk mempersempit pencarian
  3. Gunakan kotak pencarian untuk mencari data berdasarkan nama, NIK, atau desa
  4. Gunakan pagination untuk navigasi halaman data

  1. Buka menu Backup & Restore
  2. Untuk backup, pilih format file yang diinginkan (JSON, Excel, atau PDF)
  3. Untuk restore, pilih file backup JSON yang sebelumnya telah diunduh
  4. Klik Restore Data untuk mengimpor data
  5. Perhatian: Restore akan menimpa semua data yang ada
Pertanyaan Umum

Buka menu Pengaturan > Pengaturan Keamanan. Masukkan kode keamanan baru dan konfirmasi, lalu klik Simpan Kode Keamanan.

Hubungi developer aplikasi untuk reset kode keamanan. Informasi kontak developer tersedia di sidebar menu.

Buka menu Pengaturan > Daftar Desa. Tambahkan nama desa yang diinginkan, dipisahkan dengan koma, lalu klik Simpan Daftar Desa.
