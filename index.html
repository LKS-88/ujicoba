Tentang Aplikasi BARJAS

Informasi Aplikasi

BARJAS (Bantuan dan Pendataan) adalah aplikasi untuk mengelola data bantuan dari Dinas Peternakan dan Perikanan Kabupaten Situbondo.

Aplikasi ini dikembangkan untuk memudahkan dalam:

  • Pendataan penerima bantuan
  • Monitoring distribusi bantuan
  • Analisis statistik bantuan
  • Penyimpanan data di Google Drive
  • Ekspor data ke format PDF dan Excel
Informasi Teknis

Versi Aplikasi: 2.0.0

Dibangun dengan:

  • HTML5, CSS3, JavaScript (ES6+)
  • Bootstrap 5.3.0
  • Font Awesome 6.4.0
  • Chart.js untuk visualisasi data
  • jsPDF dan jsPDF-AutoTable untuk ekspor PDF
  • SheetJS untuk ekspor Excel

Penyimpanan Data: LocalStorage Browser

Keamanan Data

Aplikasi ini dilengkapi dengan sistem keamanan untuk melindungi data sensitif:

  • Kode keamanan diperlukan untuk mengakses aplikasi
  • Pembatasan percobaan login (maksimal 5 kali)
  • Pemblokiran sementara setelah percobaan gagal berulang
  • Data tersimpan di perangkat pengguna (localStorage)
Peringatan: Data akan hilang jika cache browser dibersihkan. Selalu lakukan ekspor data sebagai cadangan.
Panduan Penggunaan

  1. Buka tab "Tambah Data"
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Klik tombol "Dapatkan Lokasi" untuk mengisi koordinat secara otomatis
  4. Klik "Simpan Data" untuk menyimpan

  1. Buka tab "Data Bantuan"
  2. Gunakan filter jika perlu mencari data tertentu
  3. Klik tombol "Ekspor PDF" atau "Ekspor Excel"
  4. Untuk mencetak data tertentu, klik tombol "Cetak" pada data yang ingin dicetak

  1. Buka tab "Link Google Drive"
  2. Isi label dan URL Google Drive
  3. Klik "Simpan Link"
  4. Untuk melihat semua link yang tersimpan, scroll ke bawah
