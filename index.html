Pusat Bantuan

Panduan Penggunaan

  1. Buka menu Formulir Input
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Klik tombol Generate untuk membuat kode validasi unik
  4. Klik Simpan Data untuk menyimpan data

Pada menu Data Penerima, gunakan tombol:

  • Export ke Excel: Untuk mengekspor data ke format Excel
  • Export ke PDF: Untuk mengekspor data ke format PDF
  • Cetak Data: Untuk mencetak data langsung ke printer

Anda juga dapat menggunakan filter untuk mencetak data tertentu.

Pada menu Data Penerima, Anda dapat menggunakan berbagai filter:

  • Filter Desa: Menyaring data berdasarkan desa
  • Filter Jenis Bantuan: Menyaring berdasarkan jenis bantuan
  • Filter Tanggal: Menyaring data dalam rentang tanggal tertentu
  • Pencarian: Mencari data berdasarkan kata kunci

Klik Terapkan Filter untuk menerapkan filter yang dipilih.
