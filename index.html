Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Pastikan NIK terdiri dari 16 digit angka
  4. Pastikan nomor WhatsApp diawali dengan kode negara (contoh: 628123456789)
  5. Klik tombol Simpan Data untuk menyimpan

Anda dapat mencari dan memfilter data di menu Data Bantuan dengan cara:

  • Gunakan kotak pencarian untuk mencari berdasarkan nama, NIK, atau alamat
  • Gunakan dropdown filter untuk memfilter berdasarkan jenis bantuan, kecamatan, atau desa
  • Gunakan input tanggal untuk memfilter berdasarkan tanggal tertentu
  • Klik tombol Filter untuk menerapkan filter
  • Klik tombol Reset untuk menghapus semua filter

  1. Buka menu Laporan
  2. Pilih jenis laporan yang ingin dibuat
  3. Isi parameter laporan sesuai kebutuhan
  4. Pilih format laporan (PDF atau Excel)
  5. Klik tombol Unduh Laporan untuk mengunduh
FAQ (Pertanyaan Umum)

Untuk melakukan backup data, buka menu Pengaturan lalu klik tombol Backup Data. Data akan diunduh dalam format JSON.

Jika lupa password, hubungi administrator sistem untuk mereset password Anda.

Untuk mengubah tema, buka menu Pengaturan lalu pilih tema yang diinginkan dari dropdown Pilih Tema dan klik tombol Terapkan Tema.
