Panduan Penggunaan

FAQ

Pergi ke tab Input Data, isi formulir yang disediakan, lalu klik tombol Simpan Data.

Pada tab Data Penerima, gunakan filter yang tersedia di bagian atas tabel untuk menyaring data berdasarkan desa, status, atau kata kunci.

Pergi ke tab Laporan, atur filter sesuai kebutuhan, lalu klik tombol Export PDF atau Export Excel.

Kode validasi digunakan untuk memastikan keaslian data dan mencegah duplikasi. Kode dapat digenerate secara otomatis dengan menekan tombol Generate.

Panduan Lengkap
1. Input Data

Isi semua field yang wajib diisi pada form input data. Pastikan data yang dimasukkan valid.

2. Kelola Data

Gunakan fitur filter, pencarian, dan pagination untuk mengelola data dengan mudah.

3. Buat Laporan

Gunakan tab Laporan untuk membuat analisis data dan mengekspor dalam berbagai format.

4. Pengaturan

Ubah kode keamanan, tema tampilan, dan pengaturan lainnya sesuai preferensi.

5. Cadangan Data

Selalu backup data secara berkala untuk mencegah kehilangan data.
