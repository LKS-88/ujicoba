Bantuan dan Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Klik tombol Simpan Data untuk menyimpan
  4. Data akan tersimpan dan muncul di tabel data

  1. Buka menu Laporan
  2. Pilih jenis laporan yang diinginkan
  3. Tentukan parameter laporan (tahun, bulan, dll)
  4. Pilih format laporan (PDF, Excel, atau cetak langsung)
  5. Klik tombol Buat Laporan

  1. Buka menu Data Bantuan
  2. Klik tombol Backup di panel kontrol
  3. Data akan diunduh dalam format JSON
  4. Simpan file backup di lokasi yang aman

FAQ (Pertanyaan Umum)

Untuk mengedit data, buka menu Data Bantuan, cari data yang ingin diedit, lalu klik tombol Edit pada kolom aksi. Setelah selesai mengedit, klik tombol Update Data untuk menyimpan perubahan.

Jika lupa kode keamanan, Anda dapat menghubungi administrator sistem untuk mereset kode keamanan. Setelah direset, Anda dapat membuat kode keamanan baru di menu Pengaturan > Keamanan.

Gunakan fitur filter di menu Data Bantuan untuk memfilter data yang ingin dicetak. Setelah data difilter, klik tombol Cetak di panel kontrol. Hanya data yang terfilter yang akan dicetak.
