Panduan Penggunaan Aplikasi

Tentang Aplikasi BARJAS

Aplikasi BARJAS (Sistem Informasi Bantuan Sosial) adalah aplikasi yang dirancang untuk membantu dalam pengelolaan data bantuan sosial. Aplikasi ini memudahkan dalam:

  • Mencatat data penerima bantuan
  • Mengelola informasi bantuan yang disalurkan
  • Membuat laporan dan statistik bantuan
  • Mencetak dokumen terkait bantuan sosial

Versi: 2.1.0

Developer: Lentera karya Situbondo

Panduan Penggunaan

  1. Pilih menu Input Data
  2. Isi semua field yang diperlukan
  3. Klik tombol Simpan Data
  4. Data akan tersimpan dan muncul di tabel data bantuan

  1. Pilih menu Data Bantuan
  2. Gunakan filter jika diperlukan
  3. Klik tombol Cetak Data
  4. Pilih opsi cetak yang diinginkan

  1. Pilih menu Pengaturan
  2. Pilih tab Pengaturan Keamanan
  3. Masukkan kode baru dan konfirmasi
  4. Klik tombol Simpan Perubahan
