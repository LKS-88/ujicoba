Bantuan dan Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (bertanda *)
  3. Klik tombol Simpan Data
  4. Data akan tersimpan dan tampil di menu Data Penerima

  1. Buka menu Pencarian
  2. Isi salah satu atau beberapa field pencarian
  3. Klik tombol Cari Data
  4. Hasil pencarian akan ditampilkan di tabel bawah

  1. Buka menu Backup/Restore
  2. Pilih format backup (Excel, JSON, atau PDF)
  3. Klik tombol backup sesuai format yang diinginkan
  4. File backup akan terdownload secara otomatis
